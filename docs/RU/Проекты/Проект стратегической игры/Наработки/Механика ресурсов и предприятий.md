
## 1. Общие положения

Настоящий документ описывает систему материальных ресурсов и производственных предприятий, лежащую в основе экономической симуляции проекта "Operation Ozverin". Система обеспечивает:

- Производство, распределение и потребление тринадцати базовых ресурсов;
- Функционирование промышленных предприятий, преобразующих сырьё в продукцию;
- Связь между гражданскими специалистами и производственным сектором через требования к персоналу;
- Формальную основу для JSON-конфигураций всех производственных сущностей.

### 2.1. Классификация продукции

Продукция, производимая предприятиями, делится на три категории:

- **Мирная продукция** — предназначена для небоевых нужд: питание, одежда, строительные материалы, потребительские товары.
- **Продукция двойного назначения** — может одинаково успешно использоваться и для боевых, и для небоевых нужд: сталь, топливо, электроника, транспортные средства.
- **Военная продукция** — предназначена исключительно для боевых нужд: оружие, боеприпасы, военная техника.

## 3. Гражданские специалисты

Каждое предприятие требует определённого количества работников конкретных специальностей и квалификации. Специалисты не "потребляются" в процессе производства, а являются постоянным ресурсом, необходимым для функционирования предприятия. При нехватке персонала эффективность предприятия пропорционально снижается.

### 3.1. Категории специалистов

| Категория | Обозначение | Описание |
|-----------|------------|----------|
| Чернорабочие | Ч | Низкоквалифицированный физический труд |
| Техники | Тх | Специалисты с профтехобразованием |
| Технари | Тн | Инженеры, программисты, технологи |
| Белые воротнички | БВ | Бухгалтеры, делопроизводители, администраторы |
| Обслуга | Об | Работники сферы услуг, торговли |
| Учёные | Уч | Исследователи, двигающие науку |
| Силовики | Сил | Полицейские, охранники, спасатели |
| Гуманитарии | Гум | Юристы, экономисты, журналисты |

### 3.2. Уровни опытности

Каждый специалист имеет уровень опытности, влияющий на его производительность:

1. Стажёр (×0.3)
2. Практикант (×0.55)
3. Специалист (×1.0)
4. Профессионал (×1.4)
5. Эксперт (×1.9)
6. Мастер (×2.5)
7. Корифей (×3.5)
8. Гений (×5.0)

Производительность предприятия рассчитывается как средневзвешенная производительность всех занятых на нём специалистов.

## 5. Принципы функционирования

### 5.1. Региональная привязка

Каждое предприятие привязано к конкретному региону и может использовать только те ресурсы, которые доступны в этом регионе или в пределах логистической досягаемости. Логистическая досягаемость определяется наличием инфраструктурных объектов (дороги, порты, железные дороги).

### 5.2. Ресурсные пулы

Ресурсы хранятся в региональных пулах. Предприятие-производитель добавляет продукцию в пул своего региона. Предприятие-потребитель изымает ресурсы из пула своего региона. Перемещение ресурсов между регионами осуществляется инфраструктурными объектами и требует времени.

### 5.3. Эффективность производства

Эффективность предприятия определяется:
- Укомплектованностью персоналом (при нехватке — пропорциональное снижение);
- Квалификацией персонала (средневзвешенная производительность);
- Обеспеченностью ресурсами (при нехватке любого входного ресурса — простой);
- Технологическим уровнем (модификатор от научных открытий);
- Состоянием оборудования (износ, ремонт).

### 5.4. Расширение и модернизация

Каждое предприятие может быть:
- Построено с нуля (требует ресурсов, времени и строителей);
- Расширено (увеличение производственной мощности);
- Модернизировано (повышение эффективности, снижение потребления ресурсов);
- Конвертировано (перевод между мирным и военным производством для заводов двойного назначения).

## 6. Формальная запись рецепта

Каждый рецепт может быть представлен в виде кортежа:

$R = (I, O, P, C)$

где:
- \(I\) — множество входных ресурсов с количествами: \(\{(r_1, q_1), (r_2, q_2), ...\}\)
- \(O\) — множество выходных продуктов с количествами: \(\{(p_1, q_1), ...\}\)
- \(P\) — множество требуемого персонала по категориям: \(\{(cat_1, count_1), ...\}\)
- \(C\) — особые условия (технологические требования, географические ограничения)
